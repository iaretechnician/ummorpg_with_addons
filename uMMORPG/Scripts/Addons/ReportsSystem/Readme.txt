Install :

	1) open player prefab and in PlayerAddonsConfigurator assign scriptable configuration
	2) look in scriptable configuration and assign all needed references (SlotChatMessageInfo or SlotChatMessageInfoTMP if you use CompleteChat)
	3) Add prefab UI_Reports prefab in canvas
	4) Add ButtonReports in Shorcuts
	5) Add events onclick in ButtonReports, assign UI_Reports prefab, add action UI_Reports.ActivatePanel
